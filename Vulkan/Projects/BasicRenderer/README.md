# Vulkan BasicRenderer - 基础渲染器

这是一个使用Vulkan实现的基础渲染器项目，展示了Vulkan的核心渲染流程和功能。

## 项目概述

本项目实现了一个简单的Vulkan渲染器，主要功能包括：
- GLFW窗口创建和管理
- Vulkan实例初始化
- 物理设备和逻辑设备管理
- 窗口表面创建
- 验证层和调试信使设置
- 着色器程序加载和管理
- 基础图形渲染支持

## 目录结构

```
BasicRenderer/
├── CMakeLists.txt      # 项目编译配置
├── README.md           # 项目说明文档
├── shaders/            # 着色器文件目录
│   ├── vertex.glsl     # 顶点着色器
│   └── fragment.glsl   # 片段着色器
└── src/                # 源代码目录
    └── main.cpp        # 主程序文件
```

## 核心功能

1. **窗口管理**
   - 使用GLFW创建800x600的窗口
   - 禁用OpenGL上下文创建（纯Vulkan）
   - 窗口关闭和ESC键退出支持

2. **Vulkan初始化**
   - Vulkan实例创建
   - 验证层和调试信使设置
   - 窗口表面创建
   - 物理设备选择
   - 逻辑设备创建

3. **设备管理**
   - 物理设备能力检测
   - 队列族选择
   - 图形队列和呈现队列创建

4. **基础架构**
   - 自定义Vulkan工具类封装
   - 错误检查和异常处理
   - 资源清理机制

## 编译和运行

### 编译步骤

```bash
# 创建构建目录
mkdir build
cd build

# 配置项目
cmake ..

# 编译项目
make

# 运行程序
./bin/basic_renderer
```

### 依赖要求

- Vulkan 1.3+
- GLFW 3.x
- GLM 0.9.x
- CMake 3.10+
- C++17

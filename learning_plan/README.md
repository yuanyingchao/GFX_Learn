## 运行本地示例

要运行本地示例程序，请按照以下步骤操作：

1. 打开终端，进入 opengl_basics 目录：
   ```bash
   cd /home/sean/Code/GFX_Learn/opengl_basics
   ```

2. 确保已安装必要的依赖（GLFW、GLEW 等）。

3. 编译示例程序（如果尚未编译）：
   ```bash
   make example01_window  # 替换为你要编译的示例名称
   ```

4. 设置 DISPLAY 环境变量（如果在无图形界面环境下）：
   ```bash
   export DISPLAY=:0
   ```

5. 运行示例程序：
   ```bash
   ./example01_window  # 替换为你要运行的示例名称
   ```

## 常见问题

**Q: 运行程序时出现 "GLFW Error 65544: X11: The DISPLAY environment variable is missing" 错误？**
A: 这是因为缺少 DISPLAY 环境变量。请先设置环境变量：`export DISPLAY=:0`，然后再运行程序。

**Q: 编译示例程序时出现依赖错误？**
A: 请确保已安装所有必要的依赖，如 GLFW、GLEW 等。可以使用包管理器安装：
   ```bash
   sudo apt-get install libglfw3-dev libglew-dev
   ```

**Q: 如何查看示例程序的源代码？**
A: 示例程序的源代码位于 `/home/sean/Code/GFX_Learn/opengl_basics/` 目录下，文件名格式为 `exampleXX_xxx.cpp`。

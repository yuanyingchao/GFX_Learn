# OpenGL BasicRenderer - 基础渲染器

这是一个使用OpenGL实现的基础渲染器项目，展示了OpenGL的核心渲染流程和功能。

## 项目概述

本项目实现了一个简单的OpenGL渲染器，主要功能包括：
- GLFW窗口创建和管理
- OpenGL 3.3上下文初始化
- GLEW扩展加载
- 着色器程序加载和管理
- 基础图形渲染（三角形）
- 矩阵变换（模型、视图、投影）
- 深度测试

## 目录结构

```
BasicRenderer/
├── CMakeLists.txt      # 项目编译配置
├── README.md           # 项目说明文档
├── shaders/            # 着色器文件目录
│   ├── vertex.glsl     # 顶点着色器
│   └── fragment.glsl   # 片段着色器
└── src/                # 源代码目录
    └── main.cpp        # 主程序文件
```

## 核心功能

1. **窗口管理**
   - 使用GLFW创建800x600的窗口
   - 设置OpenGL 3.3核心配置文件
   - 实现窗口关闭和ESC键退出

2. **渲染管线**
   - 顶点缓冲对象(VBO)管理
   - 顶点数组对象(VAO)配置
   - 顶点属性指针设置

3. **着色器系统**
   - 着色器程序类封装
   - 着色器文件加载
   - 编译和链接检查
   - Uniform变量设置

4. **图形渲染**
   - 彩色三角形渲染
   - 旋转动画效果
   - 3D变换矩阵应用

## 编译和运行

### 编译步骤

```bash
# 创建构建目录
mkdir build
cd build

# 配置项目
cmake ..

# 编译项目
make

# 运行程序
./bin/basic_renderer
```

### 依赖要求

- OpenGL 3.3+
- GLFW 3.x
- GLEW 2.x
- GLM 0.9.x
- CMake 3.10+
